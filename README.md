# AdventureEngine
A Javascript Text Based Adventure Engine

## Features

- Multiple Storys
- Events (Random and Triggered)
- Items
- Rooms
- Combat
- Saves (State and export/import saves)
- Riddles
- Loot


## Storys

- Lost Island (In Progress)
- Riddle Me This (Planned)
- The Maze (Planned)
